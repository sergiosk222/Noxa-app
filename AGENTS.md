# Noxa — Codex Instructions

Noxa is a cross-platform mobile app for iOS and Android.

## Product
Noxa is a premium car-culture social app for drivers, crews, events, garage profiles, and live map interaction.

## MVP
- Registration / login
- Profile
- Garage
- Events
- Crews
- Live Map
- Privacy modes

## Design
- Dark premium UI
- Black / graphite background
- Neon red accents
- GTA / Need for Speed / The Crew energy
- Apple-level clean layout

## Bottom navigation
Crews / Events / Map / Garage / Profile

Map is the main center tab and should be visually emphasized.

## Forbidden
Do not build:
- police map
- speed camera map
- evasion features
- illegal race organizer
- public top-speed leaderboard on public roads
- public route sharing

## Coding rules
- Use TypeScript
- Keep components small
- Keep screens separate
- Do not add unnecessary libraries
- Do not rewrite the whole project unless asked
- Do not add backend yet
- Do not add real map yet
